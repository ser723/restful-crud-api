# restful_crud_api
## Description
Simple CRUD API for Bird Objects with JPA (Hibernate)

### Version
1.0.0

## Installation
- Get the project
    -clone
        ```
    git clone https://github.com/uncg-CSC340-01/restful_crud_api.git
        ```
    -OR download zip.
- Open the project in VS Code.
- This project is built to run with jdk 21.
- [Dependencies][def]

2. ### [`/{birdId}`](http://localhost:8080/birds/10) (GET)
Gets an individual Bird in the system. Each Bird is identified by a numeric `birdId`

#### Parameters
- Path Variable: `birdId` &lt;Long &gt; - REQUIRED

#### Response - A single Bird

```
  {
    "birdId": 10,
  "name": "Northern Cardinal",
  "description": "Female; buffy-brown with red accents.",
  "species": "Cardinalis cardinalis",
  "lifeSpan": 5.0
  }
```

3. ### [`/name`](http://localhost:8080/birds/name?key=Blue) (GET)
Gets a list of birds with a name that contains the given string.

#### Parameters
- query parameter: `search` &lt; String &gt; - REQUIRED

#### Response - A JSON array of Bird objects.

```
[
  {
    "birdId": 11,
    "name": "Eastern Bluebird",
    "description": "Medium-sized, vibrant blue plumage.",
    "species": "MaCyanocitta cristata",
    "lifeSpan": 7.0
  },
  {
    "birdId": 13,
    "name": "Blue Jay",
    "description": "Bright blue back, a blue crest, a white belly, and black and white markings on their wings and tail.",
    "species": "Cyanocitta Cristata",
    "lifeSpan": 7.0
  }
]
```

4. ### [`/species/{species}`](http://localhost:8080/birds/species/Cardinalis%20cardinalis) (GET)
Gets a list of birds for a named species.

#### Parameters
- query parameter: `species` &lt; String &gt; - REQUIRED

#### Response - A JSON array of Student objects.

```
[
   {
    "birdId": 10,
    "name": "Northern Cardinal",
    "description": "Female; buffy-brown with red accents.",
    "species": "Cardinalis cardinalis",
    "lifeSpan": 5.0
  }
]
```
5. ### [`/lifespan`](http://localhost:8080/birds/lifespan?lifespan=7.0) (GET)
Gets a list of birds with a lifespan meeting the Threshold.

#### Parameters
- query parameter: `lifespan` &lt;Double&gt; - REQUIRED

#### Response - A JSON array of Bird objects.

```

    "species": "MaCyanocitta cristata",

  }

```
6. ### [`/`](http://localhost:8080/birds) (POST)
Create  a new bird entry

#### Request Body
A bird object. Note the object does not include an ID as this is autogenerated.
```
{
  "name": "Ruby-Throated Hummingbird",
  "description": "A small migratory bird with iridescent green upperparts and white underparts.",
  "species": "Archilochus Colubris",
  "lifeSpan": 5.0
}
```
#### Response - The newly created Bird.

```
  {
    "birdId": 16,
    "name": "Ruby-Throated Hummingbird",
    "description": "A small migratory bird with iridescent green upperparts and white underparts.",
    "species": "Archilochus Colubris",
    "lifeSpan": 5.0
  }
```

7. ### [`/{birdId}`](http://localhost:8080/birds/10) (PUT)
Update an existing Bird.

#### Parameters
- Path Variable: `birdId` &lt;integer&gt; - REQUIRED

#### Request Body
A bird object with the updates.
```
{
  "birdId": 10,
  "name": "Northern Cardinal",
  "description": "Female; buffy-brown with red accents!!",
  "species": "Cardinalis cardinalis",
  "lifeSpan": 5.0
}
```
#### Response - the updated Bird object.
```
{
  "birdId": 10,
  "name": "Northern Cardinal",
  "description": "Female; buffy-brown with red accents!!",
  "species": "Cardinalis cardinalis",
  "lifeSpan": 5.0
}
```
